# 获取消息
- 接口
    > GET&POST /messages
- 请求参数

    **Query/表单**:

    |     参数      | 类型  | 必填 |   说明   |
    |:-----------:|:---:|:--:|:------:|
    |    token    | str | 是  |  用户令牌  |

- 返回
    **JSON:**

    |  参数   | 类型  | 必返 |   说明   |
    |:-----:|:---:|:--:|:------:|
    |  msg  | str | 是  |  返回消息  |
    | code  | int | 是  | 返回状态码  |
    | state | str | 是  | 请求是否成功 |
    | data  | map | 否  |  详见下方  |

    **data:**
    
    当上方 `state` 为 `success` 返回
    
    |   参数    | 类型  | 必返 |      说明      |
    |:-------:|:---:|:--:|:------------:|
    |   id    | int | 是  |     消息id     |
    | userid  | int | 是  |     详见下方     |
    | content | str | 是  |     消息内容     |
    | quanti  | str | 是  | 无用参数 可能在后续删除 |
    |  time   | str | 是  |   消息的发送时间    |

    **userid消息类型**

    当为0时 该消息是公开消息

    当不为0时 该消息是私有消息
