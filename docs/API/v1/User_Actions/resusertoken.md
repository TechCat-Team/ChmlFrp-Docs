# 重置令牌
- 接口
    > /resusertoken.php
- 请求参数

    Query/表单:

    |    属性     |   类型   |  说明  |
    |:---------:|:------:|:----:|
    | usertoken | string | 用户令牌 |
- 返回

    通常为两种情况 请注意分辨

    - 查询成功

        |    属性    |   类型   |   说明   |
        |:--------:|:------:|:------:|
        | newToken | string | 新的用户令牌 |
    - 查询失败

        |  属性   |   类型   |  说明  |
        |:-----:|:------:|:----:|
        | error | string | 错误信息 |
