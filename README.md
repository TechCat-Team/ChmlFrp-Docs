## 🚀 快速开始

### 前置要求
- [Node.js](https://nodejs.org/) 18+
- [pnpm](https://pnpm.io/) 8+

### 本地开发
```bash
# 安装依赖
pnpm install

# 启动开发服务器
pnpm docs:dev

# 构建生产版本
pnpm docs:build

# 预览生产构建
pnpm docs:preview
```
